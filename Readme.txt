Eiffel eTokioCabinet wrapper is based on http://1978th.net/tokyocabinet/

This library (wrapper is under development).

REQUIREMENTS
============

1)Install tokyo cabinet and dependencies
$ sudo apt-get install zlib1g-dev libbz2-dev
$ Download Tokyo Cabinet http://1978th.net/tokyocabinet/
$ tar -xvf tokyocabinet-1.4.43.tar.gz

Configure and install:

$ ./configure
$ make
$ sudo make install
